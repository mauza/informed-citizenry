# Informed Citizenry

A monorepo containing all components of the Informed Citizenry project - a platform designed to help citizens stay informed through modern technology.

## Project Structure

- `informed_citizenry_app/` - Flutter application for Android, iOS, and Web platforms
- `backend/` - Server-side applications and APIs *(coming soon)*
- `jobs/` - Background processing and ML/LLM tasks *(coming soon)*

## Components

### Mobile/Web Application
A cross-platform Flutter application that serves as the main user interface. Features include:
- User authentication and profile management
- Personalized content delivery
- Cross-platform compatibility (iOS, Android, Web)

### Backend Services *(Planned)*
- REST APIs for client applications
- Data processing and storage
- Integration with external services

### Processing Jobs *(Planned)*
- Content summarization using LLMs
- Data aggregation and analysis
- Automated content processing
