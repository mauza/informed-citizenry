## Success Metrics

We'll measure success by:

- **User Engagement**: Monthly active users, session duration, pages per session
- **Civic Impact**: Sentiments recorded, bills shared, representatives contacted
- **Data Quality**: AI summary accuracy, data freshness, coverage completeness
- **Accessibility**: Page load times, mobile usage, screen reader compatibility
- **User Satisfaction**: NPS scores, feature requests, support tickets
